# ChInput - компонент для input и textarea

### Props
| Название      | Тип                  | Описание                                                              |
|---------------|----------------------|-----------------------------------------------------------------------|
| disabled      | Boolean              | Optional. Заблокрован ли input                                        |
| clearable     | Boolean              | Optional. Можно ли очищать input                                      |
| modelValue    | String               | Optional. Значение инпута                                             |
| label         | String               | Optional. Текс лейбла над инпутом                                     |
| component     | 'input' || 'textarea'| Default = 'input'. Тип элемента. Может быть либо input либо textarea  |
| type          | String               | Default = 'text'. Тип input                                           |
| placeholder   | String               | Optional. Текст placeholder                                           |
| outerLabel    | String               | Optional. Текст внешнего лейбла                                       |
| afterText     | String               | Optional. Текст после инпута                                          |
| hasError      | Boolean              | Optional. Нужно ли отображать стили для ошибки в инпуте               |
