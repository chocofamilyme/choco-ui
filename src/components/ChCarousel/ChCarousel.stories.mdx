# ChCarousel

Компонент использует [@glidejs/glide](https://glidejs.com/) для реализации логики карусели/слайдера

## Использование

```html
<template>
  <ChCarousel>
    <ChCarouselCardItem>
      <template #left>Left content</template>
      <span>Right content</span>
    </ChCarouselCardItem>
    <ChCarouselCardItem>
      <template #left>Left content</template>
      <span>Right content</span>
    </ChCarouselCardItem>
    <ChCarouselItem>
      <div>General content item</div>
      <SomeComponent />
    </ChCarouselItem>
    <template #bullets>
      <ChCarouselBullets />
    </template>
  </ChCarousel>
</template>

<script setup>
import {
  ChCarousel,
  ChCarouselItem,
  ChCarouselCardItem,
  ChCarouselBullets
} from 'choco-ui'
</script>
```

## Компоненты

### ChCarousel

Компонент - контейнер для элементов карусели

#### Props
| Название          | Тип                                                | Описание                                                                       |
|-------------------|----------------------------------------------------|--------------------------------------------------------------------------------|
| carouselConfig    | [Glide.Options](https://glidejs.com/docs/options/) | Конфиг для карусели                                                            |

#### Slots
| Слот          | Описание                                                                                                                                                                                                                     |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| default       | Слот для отображения элементов в карусели. Элементы должны быть обернуты в компонент ChCarouselCardItem                                                                                                                      |
| bullets       | Слот для отображения точек-навигации в карусели. Для отображения есть компонент ChCarouselBullets                                                  |

### ChCarouselItem

Компонент - общий элемент слайдера без ui, который может использоваться во всех случаях

### ChCarouselCardItem

Компонент - элемент слайдера со стилями для отображения карточки с левым и правым слотами

#### Slots
| Слот          | Описание                                                                       |
|---------------|--------------------------------------------------------------------------------|
| default       | Слот для отображения правого контента                                          |
| left          | Слот для отображения левого контента                                           |

### ChCarouselBullets

Компонент - контейнер с точками-навигации
